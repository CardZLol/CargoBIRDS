# Bureau of Interorbital Resupply and Delievery.

A KSP Mod that adds parts based on the Cygnus spacecraft.

Resurfaced and its dependencies are REQUIRED For this mod to work.
Resurfaced can be downloaded here: https://github.com/Tantares/Resurfaced/releases/tag/v.1.3.1
